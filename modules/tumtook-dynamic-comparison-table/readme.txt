Dynamic comparison table builder for WordPress Pages.

== Description ==

Tumtook Dynamic Comparison Table adds a Page meta box for creating product comparison tables with flexible rows, columns, cell values, mobile tabs, shortcode output, and a dynamic Gutenberg block.

== Usage ==

Edit a Page, open "ตารางเปรียบเทียบสินค้า", enable the table, add rows and columns, then place:

[tumtook_comparison]

To render another Page table:

[tumtook_comparison page_id="123"]

== Uninstall ==

Data is preserved by default. Enable "ลบข้อมูลทั้งหมดเมื่อถอนการติดตั้ง" under Settings > Tumtook Comparison before uninstalling if you want all table meta removed.
